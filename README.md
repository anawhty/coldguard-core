# coldguard-core
ColdGuard é um sistema IoT de missão crítica para monitorar cargas ultrafrias (vacinas). Usa sensores ESP32, comunicação BLE/5G e satélite contra zonas de sombra. Possui travas de código FIFO contra estouro de memória, Watchdog por hardware e detecção em nuvem contra ataques de Jamming.
